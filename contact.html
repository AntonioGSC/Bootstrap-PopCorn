Sobre o Site

Projeto que visa a construção de uma página para indicações de filmes, séries e animes. Feito por Antonio, Loruama e Otavio utilizando Bootstrap e outras ferramentas. A página index visa uma apresentação do página de forma que o usuário se sinta acolhido, a página de contato é uma página para recomendações de filmes e para receber as recomendações no email e a de midia visa a disponibilidade de diversos filmes com suas pequenas resenhas para que o usuário escolha qual filme, série ou anime ele assistirá.
